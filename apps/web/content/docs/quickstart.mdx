This guide walks you through deploying your own While instance.

## Prerequisites

Before starting, you'll need:

<Steps>
<Step>

### Vercel Account

Sign up at [vercel.com](https://vercel.com) (free tier is sufficient)

</Step>

<Step>

### Google Cloud Project

Create a project at [console.cloud.google.com](https://console.cloud.google.com) and select **External** user type

</Step>

<Step>

### Notion Integration

Create an integration at [notion.so/my-integrations](https://www.notion.so/my-integrations)

</Step>
</Steps>

## Step 1: Create Google OAuth Credentials

1. [Go to **APIs & Services** > **Library**](https://console.cloud.google.com/apis/library) in Google Cloud Console
2. Enable the [Google Calendar API](https://console.cloud.google.com/apis/library/calendar-json.googleapis.com)

<Callout type="info">
If the API is already enabled, you'll see "Manage" instead of "Enable". No action needed.
</Callout>

3. [Go to **APIs & Services** > **OAuth consent screen**](https://console.cloud.google.com/auth/clients/create)
4. Configure as **External** with your email as test user
5. Go to **Credentials** > **Create Credentials** > **OAuth client ID**
6. Select **Web application** and configure:

**Authorised JavaScript origins:**
```
https://your-app.vercel.app
http://localhost:3000
```

**Authorised redirect URIs:**
```
https://your-app.vercel.app/api/auth/callback/google
http://localhost:3000/api/auth/callback/google
```

<Callout type="info">
You'll update these URIs with your actual Vercel domain after deployment.
</Callout>

[Detailed Google OAuth Setup →](/docs/setup/google)

<Callout type="warn" title="Token Expiration">
OAuth tokens expire every 7 days in "Testing" mode. You'll need to re-authenticate weekly unless you [publish your OAuth app](/docs/setup/google#step-5-publish-app-optional). Publishing is recommended for a smoother experience.
</Callout>

## Step 2: Create Notion Integration

1. [Go to **My integrations**](https://www.notion.so/my-integrations) in Notion
2. Click **New integration**
3. Fill out name, select workspace, keep type as **Internal**
4. Click **Save**
5. Click **Configure integration settings** in the popup
6. Verify **Read**, **Update**, and **Insert** capabilities are enabled
7. Copy the **Internal Integration Secret**
8. Connect the integration to your calendar database

[Detailed Notion Setup →](/docs/setup/notion)

## Step 3: Deploy to Vercel

Click the button below to deploy:

<a href="https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fmcsdevv%2Fwhile-dashboard-template&env=NEXTAUTH_SECRET,GOOGLE_CLIENT_ID,GOOGLE_CLIENT_SECRET,AUTHORIZED_EMAILS&envDescription=Required%20environment%20variables%20for%20While&envLink=https%3A%2F%2Fwhile.so%2Fdocs%2Fsetup%2Fvercel&project-name=while&repository-name=while">
  <img src="https://vercel.com/button" alt="Deploy with Vercel" />
</a>

During deployment, you'll need to provide these environment variables:

<EnvVarsTable vars={[
  { name: "NEXTAUTH_SECRET", description: "Random string for session encryption", generate: "base64" },
  { name: "GOOGLE_CLIENT_ID", description: "From Google OAuth credentials" },
  { name: "GOOGLE_CLIENT_SECRET", description: "From Google OAuth credentials" },
  { name: "AUTHORIZED_EMAILS", description: "Comma-separated emails, spaces trimmed (e.g., you@gmail.com, admin@company.com)" },
]} />

Redis storage is automatically configured when you add the Upstash integration from the Vercel Marketplace during deployment.

<Callout type="info" title="Generate secrets">
```bash
openssl rand -base64 32
```
</Callout>

[Detailed Deployment Guide →](/docs/setup/vercel)

## Step 4: Complete Setup Wizard

1. Open your deployed app URL
2. Follow the setup wizard to:
   - Sign in with Google (grants calendar access automatically)
   - Select your calendar
   - Enter your Notion integration token
   - Select your database
   - Configure field mappings
   - Test the connection

## Step 5: Update OAuth Redirect URI

After deployment, update your Google OAuth credentials with your actual Vercel domain:

```
https://your-app-name.vercel.app/api/auth/callback/google
```

## You're Done!

Your events will now sync bidirectionally between Notion and Google Calendar.
